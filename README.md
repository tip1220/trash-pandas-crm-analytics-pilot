# Trash Pandas Connected Reporting Pilot

A Snowflake-based sports business analytics proof-of-concept showing how a Minor League Baseball team could connect fragmented business reports into one reporting layer.

## Project Goal

The goal is to demonstrate how connected reporting can help answer:

Which games and promotions created the most total fan value beyond attendance, and which fans or accounts should sales and marketing follow up with first?

## Core Outputs

1. Homestand Intelligence Dashboard
2. Promotion Performance Scorecard
3. CRM Follow-Up Queue

## Tools

- Snowflake
- Python
- SQL
- Tableau Public
- Excel
- GitHub
- VS Code

## Data Note

This project uses public data where available and clearly labeled synthetic data where internal business data would normally be required.

It does not claim access to internal Trash Pandas data.
